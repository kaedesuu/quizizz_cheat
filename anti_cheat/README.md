# anti_cheat
> [!note]
> - description: bypass the new wayground's `"anti-cheating"` feature.
> - usage: just install the tampermonkey script and join some room without `"anti-cheating"` feature on.

To install the script, you need to:
1. install tampermonkey
2. click on the raw link: https://github.com/kaedesuu/wayground_cheat/raw/refs/heads/master/wayground_bypass_ac.user.js
3. press `Install`

## structure
- <./index.js>: the main file, not minified.
- <./index.min.js>: the minified version of the main file
- <./wayground_bypass_ac.user.js>: use for tampermonkey userscript.
