# login_spoof
> [!note]
> - description: this script allows user to join a room/game as someone else's name even if they're already joined and played the game.
> you can join and answer the game's question and perform any actions.
> - usage: when you join the room, there's a pop-up asking for your username, input's someone else's name to join as them.

To install the script, you need to:
1. install tampermonkey
2. click on the raw link: https://github.com/kaedesuu/wayground_cheat/raw/refs/heads/master/login_spoof/wayground_spoof_login.user.js
3. press `Install`

## structure
- <./index.js>: the main file, not minified.
- <./index.min.js>: the minified version of the main file
- <./wayground_spoof_login.user.js>: use for tampermonkey userscript.
