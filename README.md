# wayground_cheat
This is my own method for cheating in `wayground`, i do not update the scripts often.

## modules
- [./anti_cheat/](./anti_cheat/): this is the method to bypass wayground's `"anti-cheating"` feature.
- [./login_spoof/](./login_spoof/): allow you to join as someone username even when they are already in the room/game. You can submit your answer as them too.
- [./blatant/](./blatant): **no longer works** since they changed their API design.
