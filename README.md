# quizizz_cheat

\
The `quizizz_cheat` main script no longer work due to new API changes. I don't have any intention on fixing it, since there will be no use to it.\
The `quizizz_cheat_login` allows to login as any users (in game only).\
The `kaede_bypass_ac` will bypass the current wayground's anti-cheat. You can switch tab without noticing teacher, and play the game without entering fullscreen mode.\
File extensions explanation:
- `.js`: The original file, no minification.
- `.prod.js`: The minified version of `.js` for saving space.
- `.tampermonkey.js` or `.user.js` is the same, use for Tampermonkey, it's also minified.
